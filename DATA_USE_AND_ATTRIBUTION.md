# Data Use And Attribution

This research repository contains cleaned and processed review text derived
from [Whiskyfun](https://www.whiskyfun.com/) in order to permit inspection and
replication of the reported analysis. Source URLs are retained in the
analytical dataset where available.

The underlying review prose is copyrighted material belonging to Whiskyfun
and/or its original authors. Inclusion of processed text in this repository
does not transfer copyright and does not claim that permission for
redistribution has been obtained. The material is retained here as an
attributed research-replication dataset for the case study.

Before submitting this project for external publication or distributing the
dataset beyond this research context, the author should obtain a documented
institutional, publisher, or permissions review of the intended use and
distribution. Requests concerning the retained text should be addressed to the
repository maintainer, who can evaluate removal or access changes without
changing the analytical claims silently.
